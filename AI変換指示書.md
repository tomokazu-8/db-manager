# 材料データ → DBカセット変換指示書（AI用）

## あなたの役割

提供された材料データ（メーカーカタログ・仕入先価格表・Excelデータ等）を、
電気工事見積アプリ（estimate-app）が読み込める形式のCSVに変換してください。

---

## 出力形式（必須）

**CSV形式**で出力してください。1行目はヘッダー行です。

```
品目名称,規格名称,単位,基準単価,原価単価,原価率,歩掛1,中分類名,カテゴリ
```

### 各列の定義

| 列名 | 型 | 説明 | 例 |
|------|----|------|-----|
| 品目名称 | 文字列 | 材料の名称。メーカー名や型番は含めない | `VVFケーブル` |
| 規格名称 | 文字列 | 規格・サイズ・型番など | `2C×2.0mm` |
| 単位 | 文字列 | m / 本 / 個 / 台 / 式 / 組 など | `m` |
| 基準単価 | 数値 | 見積に使う売単価（円）。整数 | `320` |
| 原価単価 | 数値 | 仕入単価・原価（円）。整数。不明な場合は空欄 | `240` |
| 原価率 | 数値 | 原価÷売単価の比率（0〜1の小数）。不明な場合は空欄 | `0.75` |
| 歩掛1 | 数値 | 取付人工数（人工/本・個など）。不明な場合は空欄 | `0.05` |
| 中分類名 | 文字列 | 日本語カテゴリ名（下記参照） | `電線・ケーブル` |
| カテゴリ | 文字列 | 英語カテゴリID（下記参照） | `cable` |

---

## カテゴリ分類ルール

以下の基準でカテゴリを判定してください。

| カテゴリID | 中分類名 | 対象品目の例 |
|------------|----------|-------------|
| `cable` | 電線・ケーブル | VVFケーブル、CVケーブル、CVTケーブル、IV線、CPEV、同軸ケーブル、UTPケーブル |
| `conduit` | 電線管 | PF管、VE管、FEP管、ねじなし電線管、プルボックス、ケーブルラック、ダクト、アウトレットボックス |
| `device` | 配線器具 | コンセント、スイッチ、プレート、ジョイントボックス |
| `box` | ボックス | アウトレットボックス、プールボックス、スイッチボックス |
| `panel` | 分電盤 | 分電盤、住宅用分電盤、動力分電盤、開閉器盤、制御盤 |
| `fixture` | 照明器具 | LED照明、ダウンライト、ベースライト、スポットライト、非常灯、誘導灯 |
| `dimmer` | 調光器 | 調光器、ディマースイッチ |
| `fire` | 火災報知器 | 感知器、発信機、受信機、音響装置、自火報設備 |
| `ground` | 接地 | 接地銅棒、接地線、アース板 |
| `accessories` | 付属品 | サドル、バインド線、コネクタ、ブッシング、テープ類、その他副材 |

---

## 変換ルール

### 品目名称・規格名称の整理
- メーカー名（パナソニック・東芝・三菱等）は品目名称に含めない
- 型番は規格名称に記載する
- 規格（サイズ・芯数・太さ等）は規格名称にまとめる
- 例：「パナソニック WK1001 コンセント1口 15A 100V」
  → 品目名称：`コンセント（1口）`、規格名称：`15A 100V WK1001`

### 単価の扱い
- 単価は**円・整数**で記載（小数点以下切捨て）
- 税抜き価格を使用（税込みの場合は1.1で割って税抜きに変換）
- 基準単価が不明な場合は原価÷0.75を目安に計算

### 原価率の扱い
- 原価率 = 原価単価 ÷ 基準単価
- 0〜1の小数で記載（例：75%なら `0.75`）
- 不明な場合は空欄のまま

### 歩掛の扱い
- 取付人工数が不明な場合は空欄のまま（estimate-appが自動計算する）

### 除外してよい行
- 単価が0または不明で、品目名称も不明確な行
- 明らかに材料でない行（小計・合計・送料・工賃等）

---

## 注意事項

- 文字コードはUTF-8で出力してください
- カンマを含む値はダブルクォートで囲んでください（例：`"コンセント（2口、接地付き）"`)
- 数値列（単価・原価率・歩掛）に文字を入れないでください
- 1ファイル内の品目数に制限はありません

---

## 変換後の使い方

1. AIが出力したCSVを `.csv` ファイルとして保存
2. Excelで開いて「資材マスタ」シートとして保存（.xlsx）
3. DBカセットマネージャーの「ExcelからDB作成」で読み込み
4. 内容を確認・修正してExcelエクスポート
5. 見積アプリ（estimate-app）で読み込んで使用
